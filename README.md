# Multiferramenta de Pentest em Python

Bem-vindo ao projeto **Multiferramenta de Pentest em Python**! Esta ferramenta foi criada para ajudar profissionais de segurança a realizar testes de penetração de forma eficiente e organizada.

## Estrutura do Projeto

### Tools
Ferramentas práticas e utilitárias para pentests:
- **Scanner de Vulnerabilidades**: Identifica vulnerabilidades em sistemas e aplicações.
- **Reconhecimento**: 
  - Whois Lookup
  - DNS Enumeration
- **Análise de Rede**: 
  - Sniffer de Pacotes (Scapy)
  - Port Scanner
- **Exploits**: Interface para executar exploits conhecidos.
- **Fuzzing**: Testa entradas em aplicações.
- **Brute Force**: Cracker de senhas para serviços comuns (FTP, SSH).
- **Relatório**: Gerador de relatórios a partir dos resultados das ferramentas.

### Arsenal
Dividido em "Defense" e "Attack".

#### Defense
Ferramentas e técnicas para melhorar a segurança:
- **Firewall Testing**: Verifica regras de firewall.
- **Análise de Logs**: Analisa logs de segurança e detecta anomalias.
- **Monitoramento de Rede**: Monitoramento em tempo real (ex: Snort).
- **Avaliação de Configuração**: Avalia a segurança de configurações de sistemas.
- **Treinamento de Segurança**: Simuladores de ataques para treinamento.

#### Attack
Ferramentas e técnicas para simulação de ataques:
- **Social Engineering**: Simula ataques de phishing.
- **Web Application Attacks**:
  - SQL Injection Testing
  - Cross-Site Scripting (XSS) Testing
- **Network Attacks**: 
  - Simulações de DoS/DDoS.
  - Exploração de vulnerabilidades em protocolos.
- **Post-Exploitation**: Mantém o acesso após um ataque.
- **Metasploit Integration**: Integração com Metasploit para facilitar a execução de exploits.
